Social Leaderboards

OpenKit social leaderboards, combined with social challenges, let players see how they rank amongst their friends, and see how they stack up against all players.

OpenKit social leaderboards have two sections: Friends' scores, and Global Scores.

OpenKit User Accounts

OpenKit user accounts are created when a player authenticates with either a Facebook account or a Google account. This allows players to submit scores to OpenKit leaderboards.

OpenKit Leaderboards and GameCenter

OpenKit social leaderboards work in tandem with GameCenter leaderboards. When using both GameCenter and OpenKit, OpenKit will display the global leaderboard from GameCenter, and show friends scores from Facebook (OpenKit) as well as from GameCenter.

If you're using both, be sure to create OpenKit leaderboards in the OpenKit dashboard that match your defined GameCenter leaderboards. Then in the OpenKit dashboard, for each leaderboard, be sure to specify your corresponding GameCenter category

When submitting a score using the OKScore class, be sure to set the GameCenter leaderboard category. OpenKit will always submit scores to GameCenter when specified.

Unity

When using OpenKit functions, be sure to import OpenKit:

using OpenKit;

Show Leaderboards

To show a list of leaderboards, call:

OKManager.ShowLeaderboards();

Show a Specific Leaderboard

To show a specific leaderboard, call:

OKManager.ShowLeaderboard(leaderboardID);

LeaderboardID is the OpenKit leaderboard ID for the leaderboard you want to show.

Submit a Score

To submit a score, you simply create an OKScore object, set the appropriate values, then call Submit.

If the player is not authenticated with OpenKit, their score is cached locally and submitted once they authenticate.

long ScoreValue = 234234;
int SampleLeaderboardID = 23423;
OKScore score = new OKScore(ScoreValue, SampleLeaderboardID);
score.displayString = "4334 points";
score.gameCenterLeaderboardCategory = SampleLeaderboardGameCenterCategory;
//If you're using OpenKit leaderboards alongside GameCenter on iOS, be sure to set the GameCenter leaderboard Category
		
Action<bool, string> submitCallback = (success, errorMessage) => {
	if (success) {
		Debug.Log("Score submitted successfully!");
	} else {
		Debug.Log("Score did not submit. Error: " + errorMessage);
	}
};

score.SubmitScore(submitCallback);

OKUser Account

To check if a player is authenticated with OpenKit, simply call:

if(OKUser.GetCurrentUser() == null) {
	//Player is not logged in
} else {
	// Player is logged in
}

iOS

When using OpenKit functions, be sure to import OpenKit:

#import "OpenKit.h"

Show Leaderboards

To show a list of leaderboards, call:

OKLeaderboardsViewController *leaderBoards = [[OKLeaderboardsViewController alloc] init];
// Set the showLandscapeOnly property on OKLeaderboardsViewController to force landscape orientation
//[leaderBoards setShowLandscapeOnly:YES];
    
[self presentViewController:leaderBoards animated:YES completion:nil];

Show a Specific Leaderboard

To show a specific leaderboard, call:

OKLeaderboardsViewController *leaderBoards = [[OKLeaderboardsViewController alloc] initWithDefaultLeaderboardID:385];

[self presentViewController:leaderBoards animated:YES completion:nil];

LeaderboardID is the OpenKit leaderboard ID for the leaderboard you want to show.

Submit a Score

To submit a score, you simply create an OKScore object, set the appropriate values, then call Submit.

If the player is not authenticated with OpenKit, their score is cached locally and submitted once they authenticate.

OKScore *score = [[OKScore alloc] init];
[score setScoreValue:224342];
[score setDisplayString:@"224342 points"];
[score setOKLeaderboardID:385];
[score setGamecenterLeaderboardID:@"leaderboard1"];

[score submitScoreWithCompletionHandler:^(NSError *error) {
    if(error) {
        NSLog(@"There was an error submitting the score");
    } else {
        NSLog(@"Score submitted successfully");
    }
}];

if ([OKUser currentUser] == nil) {

    //Player is logged in
}

Android

When using OpenKit functions, be sure to import OpenKit:

import io.openkit.*;

Show Leaderboards

To show a list of leaderboards, call:

Intent launchOKLeaderboards = new Intent(MainActivity.this, OKLeaderboardsActivity.class);
startActivity(launchOKLeaderboards);

Show a Specific Leaderboard

To show a specific leaderboard, call:

Intent launchleaderboard = OKLeaderboard.getLeaderboardIntent(MainActivity.this, LeaderboardID);
startActivity(launchleaderboard);

LeaderboardID is the OpenKit leaderboard ID for the leaderboard you want to show.

Submit a Score

To submit a score, you simply create an OKScore object, set the appropriate values, then call Submit.

If the player is not authenticated with OpenKit, their score is cached locally and submitted once they authenticate.

OKScore score = new OKScore();
score.setScoreValue(12345);
score.setDisplayString("12345 OpenKit points");
score.setOKLeaderboardID(385);

score.submitScore(new OKScore.ScoreRequestResponseHandler() {
	
	@Override
	public void onSuccess() {
		Debug.Log("OpenKit"", "Submitted score successfully");
	}

	@Override
	public void onFailure(Throwable error) {
		Debug.Log("OpenKit"", "Failed to submit score with error: " + error);
	}
});
